About This Project

CareFlow started as a B.Tech project in 2018 — a Tkinter desktop app with SQLite for managing hospital appointments, with a text-to-speech queue announcer built in Python.

This version upgrades it to a production-grade REST API using modern tooling, ready for real clinic deployment.

Original: Tkinter + SQLite + pyttsx3
Upgraded: FastAPI + PostgreSQL + JWT + Docker

Tech Stack

FastAPI — REST API with auto Swagger docs
SQLAlchemy — ORM (SQLite / PostgreSQL)
JWT Auth — Secure doctor login (jose + passlib)
Pydantic v2 — Request/response validation
Docker — One-command deployment
Railway — Cloud deployment + managed PostgreSQL
